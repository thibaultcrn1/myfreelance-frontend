## Installation

1. Cloner le repo
   ```sh
   git clone https://github.com/thibaultcrn1/myfreelance-frontend.git
   ```
2. Installer les packages NPM
   ```sh
   npm i
   ```
3. Exécuter le fichier `index.js`
   ```sh
   pm2 start index.js
   ```
